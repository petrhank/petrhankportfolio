Symphony Box

Symphony Box je webová aplikace, která přehrává orchestrální zvukovou smyčku a umožňuje uživateli přidávat nebo odebírat jednotlivé nástroje a sekce. Uživatel tak může zkoumat, jakou roli má v dané kompozici každý nástroj a má možnost je kombinovat dle své libosti. Symphony Box je kromě projektu do mého front-end portfolia i mou skladatelskou tvorbou.

O mně

Kdo jsem?

Jmenuji se Petr. Narodil jsem se ve Vrchlabí a nyní již třetím rokem žiji v našem hlavním městě.

V květnu 2020 jsem, spíše pro zábavu, začal studovat programování webových stránek. To, co se ze začátku zdálo být koníčkem, brzy přerostlo ve směr, kterým se chci vydat na profesní dráze, a tak jsem programování začal věnovat většinu svého volného času.

Za tu dobu se mi povedlo získat solidní znalost HTML, CSS a základy JavaScriptu, ve kterém se stále zdokonaluji. V budoucnu bych se také kromě front-endu rád naučil alespoň základy back-endu.

Miluji hudbu, výlety do přírody, svou přítelkyni a naší fenku chodského psa, Aiwy.

Zkušenosti

  • HTML

    V HTML mi jde hlavně o to, aby byl kód sémantický, čitelný a aby se v něm dalo dobře orientovat.

  • CSS

    V CSS používám moderní postupy pro dosažení responzivnosti webu, rád pracuji s CSS Gridem a Flexboxem.

  • SASS

    Preprocesor, který značně usnadňuje práci s CSS a zpřehledňuje kód. Používám ho denně.

  • Javascript

    JavaScript stále studuji. Nyní umím základy, jako funkce, pole, objekty, manipulaci s DOM, a další. Stále se učím a zdokonaluji.
